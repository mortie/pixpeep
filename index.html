When doing image processing of any kind, debugging issues is often difficult. Neither GDB nor printf provides good visibility into the contents of image buffers.

PixPeep tries to make image buffers less opaque. Write your image buffer to a file, load it into PixPeep, configure width/height/pixel format/alpha mode, and see what the buffer actually contains at whatever stage in the pipeline they're from.

You can find test files to experiment here.
